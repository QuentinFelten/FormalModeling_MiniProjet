M2 Formal modeling project by Quentin FELTEN, Remy CHERKHAOUI and Daniil ROSSO
Efrei

### Logical Symbol interpretation for the input file 

* NOT → !
* AND → &
* OR → OR
* THEN → >
* IFAOF → <>
* BELONGS TO → BELONGS
* EXISTS → EXISTS
* FOR EVERY → v
* PHI → PHI
* PSI → PSI
* IS TRUE → IST 

### Glossary

* A → All
* E → Exists
* X → Next
* G → Globally
* F → Finally
* x U y → x Until y


### Input format 

* Kripke as a 4 tuple (S,I,R,L)
* CTL following the interpretation in one line
