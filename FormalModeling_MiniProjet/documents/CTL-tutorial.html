Logical Symbol interpretation for the input file

Example of a valid formula :
A(U(&(!(|(p,q)),!(&(p,q)))))

  • NOT → !(x)
  • AND y → &(x,y)
  • x OR y → |(x,y)
  • x THEN y → T(x,y)

Glossary

  • For All → A
  • Exists → E
  • Next x → E(X(x)) or A(X(x))
  • Globally x → E(G(x)) or A(G(x))
  • Finally x → E(F(x)) or A(F(x))
  • x Until y → E(U(x,y)) or A(U(x,y))
